Get Started in 3 Steps

From zero to a running personal AI agent in under 2 minutes.

1

Clone the repo

git clone https://github.com/consblock/local-agent.git cd local-agent
2

Run the setup

Interactive wizard walks you through API keys, messaging, and workspaces.

./consblock init
3

Start Agent

That's it. Your agent is running, listening, and ready to take commands.

./consblock up # Chat with your dedicated local agent ./consblock chat # Or send a one-shot payload command ./consblock run "Hire AlphaQuant for 1 hour"
